# PROJECT RULES

Always use TypeScript.

Always use Next.js App Router.

Always use TailwindCSS.

Always use shadcn/ui.

Never duplicate code.

Never use inline styles.

Always create reusable components.

Business logic must never live inside UI components.

Keep calculations inside services.

Keep components small.

Use clean folder structure.

Always use dark mode.

Animations should be subtle.

Performance first.

Accessibility first.

Never install unnecessary packages.

Use descriptive variable names.

Keep files organized.

Comment only when necessary.

Think before coding.

If an existing component can be reused,
reuse it instead of creating another one.
